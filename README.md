# MemoClaw SDKs

Official SDKs for the [MemoClaw](https://memoclaw.com) memory API.

## Available SDKs

| Language | Package | Status |
|----------|---------|--------|
| Python | [`memoclaw`](./python/) | Available |
| TypeScript | `@memoclaw/sdk` | Coming soon |

## What is MemoClaw?

MemoClaw is a semantic memory layer for AI agents. Store, recall, and relate memories with automatic deduplication, hybrid retrieval, and intelligent decay.
